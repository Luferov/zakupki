Задача

Цветовая схема: приятная, контрастная, с акцентными цветами для кнопок

Анимации: плавные переходы между страницами и hover-эффекты
Иконки: emoji или SVG для визуальной навигации
Уведомления: всплывающие сообщения при действиях (добавление в корзину, оплата)

✅ Критерии качества
Все 4 страницы переключаются без перезагрузки
Поиск и фильтры работают корректно
Добавление товара в корзину обновляет счётчик
Форма продажи добавляет товар в общий список
Корзина позволяет удалять товары и считает сумму
Форма оплаты содержит все указанные поля
Код чистый, с комментариями на русском языке
💡 Примечания для разработчика
Виджет карты: можно использовать заглушку или iframe (Яндекс.Карты / Google Maps)
Связь с продавцом: реализовать через модальное окно с выбором способа
Отзывы: можно сделать статичными или с возможностью добавления
Хранение данных: достаточно временного хранения в JS-массиве (без backend)
📌 Пример желаемого результата
После открытия файла в браузере пользователь видит главную страницу с двумя кнопками. При нажатии на «Купить товары» открывается каталог с поиском и фильтрами. Товары можно добавлять в корзину, а затем оформлять заказ с выбором адреса на карте и способа оплаты.
